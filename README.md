# How to start

 -> CREATE DATABASE !Change_Me(name_db)!; in your database
 -> Set your preferences in the .env file
 -> composer install
 -> php artisan migrate
 -> php artisan db:seed
 -> php artisan serve
 -> npm install && npm run build
 -> php artisan key:generate
 -> php artisan test
 -> Enjoy
